# FYBlog (Back End)
This project is built to support the management and api for FYBlog.
( Build my personal Blog )

## Features
* Server is rely on Express and Mongoose
* React.js + Redux + ES6
* Webpack

## Getting Started

```
npm install
npm install forever -g

// Start this app
npm run server-start

// Stop this app
npm run server-stop
```
